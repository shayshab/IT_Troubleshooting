# Printer Troubleshooting Guide

| Issue | Solution |
|-------|----------|
| Printer not detected | Check cables, power, reconnect |
| Printer not printing | Clear queue, restart devices |
| Print jobs stuck | Cancel jobs, restart spooler |
| Poor print quality | Check ink/toner, clean heads |
| Paper jams | Remove jam, reload tray |
| Printer offline | Set as default, restart |
| Driver issues | Update/reinstall drivers |
| Cannot add printer | Check permissions, restart |
| Wireless not connecting | Reconnect WiFi, restart router |
| Slow printing | Lower quality, clear jobs |
| Error codes | Check manual/support |
| Scanning not working | Update drivers, restart |
| Double-sided not working | Enable duplex, check settings |
| Color issues | Check cartridge, calibrate |
| Sharing not working | Check sharing, firewall |
